
# Meu Portfólio

Bem-vindo ao meu portfólio! Este é um site pessoal que fui responsável por desenvolver utilizando **React**. Aqui você pode conhecer um pouco sobre minhas habilidades, experiência, projetos e formas de contato.

## Tecnologias Usadas

Este portfólio foi desenvolvido utilizando as seguintes tecnologias:

- **React**: Biblioteca para construir a interface do usuário.
- **React Router**: Para navegação entre páginas.
- **Framer Motion**: Para animações e transições suaves.
- **Tailwind CSS**: Framework CSS para construção rápida e responsiva.
- **React Icons**: Para ícones interativos nas seções do site.

## Funcionalidades

- **Navegação**: O site possui links de navegação para seções como "Home", "Skills", "Projetos" e "Experiência".
- **Responsividade**: O layout é completamente responsivo, adaptando-se a diferentes tamanhos de tela.
- **Social Links**: Links interativos para as minhas redes sociais (GitHub, Instagram, Discord, etc.).

## Como Rodar o Projeto Localmente

Para rodar o projeto localmente, siga os passos abaixo:

1. Clone o repositório:

   ```bash
   git clone https://github.com/ualcz/portfolio.git
   ```

2. Navegue até o diretório do projeto:

   ```bash
   cd portfolio
   ```

3. Instale as dependências:

   ```bash
   npm install
   ```

4. Rode o servidor de desenvolvimento:

   ```bash
   npm start
   ```

5. Acesse o projeto em `http://localhost:3000`.
